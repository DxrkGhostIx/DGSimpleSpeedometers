# DG Simple Speedometers

A simple and clean speedometer script for FiveM.

## Features

- Speedometer display
- Fuel display
- Minimal and clean design
- Customizable position and size

## Commands

- `/editspeedometer` — Edit the position and size of the speedometer
- `/resetspeedometer` — Reset the speedometer to the default position and size

## Installation

1. Drag and drop the folder into your `resources` folder.
2. Add the following line to your `server.cfg`: `ensure DGSimpleSpeedometers`
